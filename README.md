## Инструкция по установке локальной LLM (без GPU) на сервер и подключению веб-интерфейса OpenWebUI

Инструкция доступна здесь: https://ekashi95.github.io/local_lmm/

Используется Ollama в связке с OpenWebUI. Это не единственный возможный вариант (и даже не самый лучший) развёртывания локальной LLM, но один из наиболее простых в освоении и популярных. 

Инструкция подготовлена для моих коллег, но буду рада, если она поможет кому-то ещё.
